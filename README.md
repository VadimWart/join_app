    <h1>Kanban Board Project</h1>
    <p>
        A responsive and interactive <strong>Kanban Board</strong> application built using 
        <code>JavaScript</code>, <code>HTML</code>, <code>CSS</code>, and 
        <code>Firebase Realtime Database</code>. This project is designed for managing tasks 
        efficiently with features like task creation, editing, and categorization.
    </p>

    <h2>‚ú® Features</h2>
    <ul>
        <li><strong>Task Management:</strong>
            <ul>
                <li>Add, edit, and delete tasks.</li>
                <li>Categorize tasks (e.g., To-Do, In Progress, Done).</li>
            </ul>
        </li>
        <li><strong>User Authentication:</strong>
            Login and signup system using Firebase Authentication.</li>
        <li><strong>Real-Time Updates:</strong>
            Synchronize tasks across users and devices with Firebase Realtime Database.</li>
        <li><strong>Mobile-Friendly Design:</strong>
            Fully responsive UI for mobile, tablet, and desktop devices.</li>
        <li><strong>Customizable:</strong>
            Easy to extend and customize the codebase for additional features.</li>
    </ul>

    <h2>üìã Usage</h2>
    <h3>1. Login or Signup</h3>
    <p>Users need to log in or create an account to access their Kanban board.</p>

    <h3>2. Add Tasks</h3>
    <ul>
        <li>Click on "Add Task" to create a new task.</li>
        <li>Assign categories and due dates.</li>
    </ul>

    <h3>3. Manage Tasks</h3>
    <ul>
        <li>Drag and drop tasks between categories.</li>
        <li>Edit or delete tasks as needed.</li>
    </ul>

    <h3>4. Real-Time Collaboration</h3>
    <p>Any changes made are immediately reflected across devices.</p>
